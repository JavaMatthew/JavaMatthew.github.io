## 作用

1. 数据副本
2. 扩展读性能

## 实现方式

### slaveof命令

```
slaveof 复制（异步）
slaveof no one 取消复制（不会清除历史数据，只是不再同步新的数据）
```

### 配置

```
slaveof ip port
slave-read-only yes
```

### 比较

|方式|命令|配置|
|--|--|--|
|优点|无需重启|统一配置|
|缺点|不便于管理|需要重启|
