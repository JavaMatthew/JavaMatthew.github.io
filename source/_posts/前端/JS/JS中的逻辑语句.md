## JS中的操作语句：判断、循环

### 判断

> 条件成立做什么? 不成立做什么?

- if/else if/ else
- 三元运算符
- switch case

1. if/else

    ```js
    if(条件){
        条件成立执行
    }else if(条件2){
        条件2成立执行
    }
    ...
    else{
        以上条件都不成立
    }
    ```

2. == VS ===

    == ：相等（如果左右两百年数据值类型不同，是默认向转换为相同的类型，然后比较）
    `'5' == 5  =>TRUE`
    === : 绝对相等（如果类型不一样，肯定不相等）
    `'5' == 5  =>FALSE`
    项目中为了保证业务的严谨，推荐使用===
