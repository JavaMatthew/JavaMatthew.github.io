## 注释

注释是给人看的，不是给计算机看的。 Java 中共有 3 种类型的注释：

1. 单行注释： 以//开头， //后面的所有内容均被当作注释处理。

2. 多行注释： 以/* 开头，以 * /结束，中间的所有内容均被当作注释处理。 多行注释来源于C/C++。 关于多行注释，<font color=red>需要注意的是，多行注释不能嵌套。</font>

3. 另一种多行注释。 用于产生 Java Doc 帮助文档。 暂且不介绍

```java
//

/*
*/

/**
*/
```
